# RealTimeBusTrack
Real-time Bus Tracking Map Line 1 of the MBTA (Massachusetts Bay Transportation Authority) between Nubian and Harvard.

# How to run
1. Obtain API key from https://developers.google.com/maps
2. Download the repo and open index.html

# Future Improvements
1. Include popup containing info such as next stop, seat availability, real-time for each bus.
2. Include static icon for each stop on the line.
